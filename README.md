实现了一个基于miniqmt和XtQuant实现的一个自动化交易软件，能够对指定的A股股票进行自动买入卖出交易。其实是从8月份开始写这个软件，用了两周就上线实盘，但是由于工作的原因，开市的时候都在上班，不开市很多bug又没法调，只能通过打印日志一点点调试，终于做到没有大的bug了，期间也交了不少学费，真金白银的学费。
目前软件的交易策略非常简单，就是高抛低吸，对某支股票以买入价A买入，然后以A+x的卖出价卖出。软件已进行实盘，目前对于一些股价波动不大的股票，例如银行，港口等，如果在一定周期内股价小幅波动，可以做到盈利，当然如果某支股票一直处于持续下跌趋势，如大秦铁路，就会导致亏损。
这个软件的目的并不是实现高额的收益，而是适用于一些养老股，通过不断的小额高抛低吸，降低成本，其目标收益是超过理财产品，进而可以达到10%~20%的年化收益。
后续可能会加入量化策略，来优化自动化交易过程。从我的计划而言，想进一步提升资金的利用率，即每天成功实现高抛低吸交易的笔数，达到有限资金获取更高的盈利。
